🛠 One-Time Setup — ~10 Minutes
1

Open Google Apps Script

Go to script.google.com → click New Project. Delete all default code in the editor.

2

Paste the Webhook Script

Download the file square-webhook.gs (provided alongside this app). Copy its entire contents and paste into the Apps Script editor. Save with Ctrl+S.

3

Run the Test Function First

In the Apps Script editor, select testSetup from the function dropdown → click ▶ Run. Accept the permissions popup. This creates your Google Sheet with the right columns and inserts a test order so you can confirm it's working.

4

Deploy as Web App

Click Deploy → New deployment → choose type Web app.
Set: Execute as: Me | Who has access: Anyone
Click Deploy → copy the Web App URL (ends in /exec).

5

Paste URL Above & Connect

Paste the Web App URL into the connection box above and click Connect & Test. Your test order should appear in the Orders tab immediately.

6

Set Up Square Webhook

Go to developer.squareup.com → your app → Webhooks.
Add endpoint: paste your Web App URL
Subscribe to events: order.created · order.updated · payment.created
From now on, every Square order triggers automatically — no action needed from you!
